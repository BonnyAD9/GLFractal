# GLFractal
[dont click me](https://www.sspu-opava.cz/static/UserFiles/File/_sablony/CJ_II/VY_32_INOVACE_D-10-12.pdf) 
App that renders fractals using OpenGL
## Supported fractals:
- Mandelbrot set
- Julia set
- Newton fractal
- Nova fractal
